## Quick Start

<Steps>
  <Step title="Register for an account">
    Sign up at `https://dashboard.ai4tza.or.tz` and verify your email.
  </Step>
  <Step title="Generate API key">
    Go to **API Keys** section and click "Create Key".
  </Step>
  <Step title="Call the API">
    ````bash
    curl -H "Authorization: Bearer YOUR_KEY" https://api.ai4tza.or.tz/v1/info
    ````
  </Step>
</Steps>

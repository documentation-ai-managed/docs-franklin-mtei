## Updating the Platform

Follow these steps to update your AI4TZA client SDK to the latest version. You ensure access to the newest features and security patches.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your terminal.

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm list ai4tza-sdk
    ````

    ````bash
    yarn list ai4tza-sdk
    ````
    </CodeGroup>
  </Step>

  <Step title="Install Latest" icon="download">
    Update to the latest version.

    <Tabs>
      <Tab title="npm" icon="npm">
      ````bash
      npm install ai4tza-sdk@latest
      ````
      </Tab>
      <Tab title="yarn" icon="yarn">
      ````bash
      yarn add ai4tza-sdk@latest
      ````
      </Tab>
    </Tabs>
  </Step>

  <Step title="Verify Update" icon="check-circle">
    Test the new inference endpoint.

    ````javascript
    import { AI4TZA } from 'ai4tza-sdk';

    const client = new AI4TZA({ apiKey: 'YOUR_API_KEY' });
    const result = await client.infer({ prompt: 'Hello, Tanzania!' });
    console.log(result);
    ````
  </Step>
</Steps>

<Callout kind="tip">
  Subscribe to updates via the dashboard at `https://dashboard.example.com/notifications` to receive email alerts for new releases.
</Callout>
